MacBook Jarvis ist ein KI-Supervisor der im Hintergrund läuft, Claude Code autonom steuert und deinen Mac vollständig kontrolliert.

Für Mac herunterladen
macOS 12+ · Kostenlos · Keine Registrierung · Quellcode ansehen
⚡

Autonomer Task-Loop

Legt Aufgaben in eine Liste — Jarvis erledigt sie automatisch mit Claude Code und bewertet das Ergebnis.

🧠

Lokale KI-Bewertung

264+ Fehlermuster erkennen ob eine Aufgabe wirklich erledigt wurde. Kein API-Key nötig.

💻

Mac-Steuerung

Öffnet Apps, schließt Tabs, wechselt Modi per Sprachbefehl. Alles per AppleScript.

🔐

Passwort-Vault

Verschlüsselt gespeichert. Erkennt automatisch wenn ein Passwort falsch ist.

🤖

Menüleisten-App

Immer griffbereit in der Menüleiste. Dashboard im Browser. Alles lokal.

🎙

Sprachsteuerung

Siri Shortcut → Jarvis wechselt den Modus. Kein Tippen, keine Maus nötig.

Erste Öffnung — macOS Sicherheitshinweis

1
DMG herunterladen und Jarvis in den Programme-Ordner ziehen
2
macOS zeigt einen Sicherheitshinweis — das ist normal bei Apps ohne Apple-Zertifikat
3
Rechtsklick auf Jarvis.app → Öffnen → Bestätigen
4
Fertig — Jarvis erscheint in der Menüleiste und startet automatisch bei jedem Login
